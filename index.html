FAQs

  • What format should the search query be in?

    No specific format is required. Just a simple phrase or sentence will suffice. As shown above, we highlight specific terms in your search query so that you know what the result of your query will look like.

  • What else can I search for?

    You can search for the following items in the current version: Player Name, Team Name, Shot Type, Shot Distance.

  • Do I need to enter team names in a particular format?

    Team names don't need to be in a particular format. For example, "POR" can be represented as "Blazers" as well as "Portland Trail Blazers".

  • What is the "Select CSV" button?

    That button is used to input CSV files instead of a natural language query. Please refer to docs.dunkengine.com for more info.
